# Usage Guide

## Deutsch
1. Anwendung starten.  
2. Master-Passwort setzen.  
3. Datei auswählen und verschlüsseln oder entschlüsseln.  
4. Alle Aktionen werden im Journal gespeichert.

<br>

---

<br>

## English
1. Launch the application.  
2. Set the master password.  
3. Select a file to encrypt or decrypt.  
4. All actions are logged in the journal.
