# Frequently Asked Questions

## Deutsch
**Was passiert, wenn ich mein Master-Passwort verliere?**  
→ Die Datenbank bleibt unzugänglich. Es ist kein Wiederherstellungsprozess möglich.

**Wie sicher ist AES-256-GCM?**  
→ Es handelt sich um eine aktuelle, von der Industrie anerkannte Verschlüsselungsmethode.

## English
**What if I lose my master password?**  
→ The database remains inaccessible. There is no recovery process.

**How secure is AES-256-GCM?**  
→ It is an industry-standard encryption method considered very secure.
